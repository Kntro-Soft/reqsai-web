# Authors — Reqs-AI Web (Frontend)

This frontend is developed by the **Kntro-Soft** team as part of the course **1ASI0732 - Emerging
Software Architecture** at Universidad Peruana de Ciencias Aplicadas (UPC), academic cycle 2026-01.

It implements the UI for the architecture defined in the [Reqs-AI Report](https://github.com/Kntro-Soft/ReqsAI-Report):
a feature-based Angular 22 SPA, consuming the Spring Boot 4 backend ([`reqsai-api`](../reqsai-api)).

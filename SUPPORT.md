# Support — Kntro-Soft / reqsai-web

## About This Repository

This repository contains the **frontend** for the Reqs-AI platform, developed for the course
**1ASI0732 - Emerging Software Architecture** at UPC. It is an academic project.

## Getting Help

### Running or building the project

Start with the documentation:

- [README.md](./README.md) — quickstart (install, run, build, test)
- [CONTRIBUTING.md](.github/CONTRIBUTING.md) — development workflow, tests, branches, commits
- [docs/adr/](./docs/adr/) — architecture decisions (the rationale behind key choices)
- [docs/DEPLOYMENT.md](./docs/DEPLOYMENT.md) — Docker and AWS S3+CloudFront deployment guide

### Questions and discussions

- [Open an Issue](https://github.com/Kntro-Soft/reqsai-web/issues/new/choose) using the appropriate template
- [GitHub Discussions](https://github.com/Kntro-Soft/reqsai-web/discussions) for general questions

## Security Issues

To report a vulnerability or accidentally exposed secret, **do not open a public Issue**. Follow the
instructions in [SECURITY.md](.github/SECURITY.md).

## Backend Repository

The API that this frontend consumes lives at
[`Kntro-Soft/reqsai-api`](https://github.com/Kntro-Soft/reqsai-api). If your issue is related to
an API endpoint or backend behavior, open the issue there instead.
